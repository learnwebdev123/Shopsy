Shopsy

E-commerce Website using modern web technologies, including React.js for UI and Tailwind CSS for styling, and the Fakestore API as the data source. <br />

Features <br />
Products Listing <br />
Filter Functionality <br />
Add to Cart <br />
Remove from Cart <br />
Checkout <br />
Responsive Design <br />

Tech Stack <br />
React.js <br />
Tailwind CSS <br />
Fakestore API <br />
React Router <br />
React Icons <br />
